# Portfólio QA — Ian Santos

Portfolio de documentação de QA containing planejamentos, casos de teste, relatórios de bugs e análises de risco para múltiplos projetos.

---

## 📂 Estrutura

```
portfolio-qa/
├── projetos/
│   ├── barber-os/      # Sistema de gestão de barbearia
│   ├── detailer-app/   # App mobile de detailing automotivo
│   ├── klipper/        # Agenda digital para barbearia
│   └── jobpilot/       # Sistema de automação de candidaturas
├── templates/          # Templates reutilizáveis para documentação QA
├── automacoes/        # Scripts de automação
├── evidencias/        # Evidências de teste (screenshots, logs)
└── SESSION_STATE.md    # Estado da sessão de documentação
```

## 📝 Templates Disponíveis

| Template | Uso |
|----------|-----|
| `plano-de-testes.md` | Estratégia e escopo de testes |
